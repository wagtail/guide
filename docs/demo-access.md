# Demo access

The guide website is configured with a private demo section, so we can easily share access to a realistic demo of [Wagtail AI](https://wagtail.org/wagtail-ai/) and [Wagtail Localize](https://wagtail.org/wagtail-localize/), with real-world content.

## Invitation email

Use this template when sharing demo access with people:

```markdown
Hi,

Our Wagtail user guide website is now ready for you to access and trial Wagtail features, in particular Wagtail AI. Your account is set up with your email address `<Email>` as the username. To access the CMS and try out the features, you need to:

1. Go to the password reset form ([guide.wagtail.org/admin/password_reset/](https://guide.wagtail.org/admin/password_reset/)), enter your email and initiate a reset.
2. Look for a “Password set” email from “Wagtail user guide” in your inbox, follow the password reset link, and set a new password.
3. Login with your new password.

If this doesn’t work, just let me know and I’ll help!

Within the CMS, you will find the AI features in those areas:

-   [Pages > Demos](https://guide.wagtail.org/admin/pages/1476/): those pages are only visible to logged-in users, feel free to change them however you wish!
-   [Images](https://guide.wagtail.org/admin/images/): to generate titles and descriptions (used as alt text). Those images are only available in the demo pages, so feel free to also upload new ones.
-   [Settings > Agents](https://guide.wagtail.org/admin/settings/wagtail_ai/agentsettings/1/): for configuration of most AI features on pages and images.
-   [Settings > Prompts](https://guide.wagtail.org/admin/prompt/): for configuration of the rich text fields integration which is still a work in progress.
-   Finally, under the “Help” menu, you can find the site’s documentation style guide, and llms.txt files - if you want to experiment with AI tools outside the CMS with the site content and guidance.

Keep in mind this is a **shared demo environment.** Your demo content will be visible to other CMS users, and any Agents or Prompts changes in particular will apply site-wide.

Please let me know if you have any questions!
```
